## About

Challenge project for BuyCoins Frontend Engineer role

### Live demo

[https://bc-challenge.netlify.app](https://bc-challenge.netlify.app)

### How to run
```
$ git clone https://github.com/tolulawson/buycoins-challenge
$ cd buycoins-challenge
$ yarn install
$ yarn run start
```

### How to build
```
$ yarn run build
```
